# Chat Relay Bot

채팅 중계용 디스코드 봇

# env

```env
BOT_TOKEN="Your Bot Token"
BOT_PRESENCE_NAME="채팅 중계"
API_URI="채팅 중계 웹서버 주소 (ex. http://localhost:3000/chat)"
```

# How to use

## Clone this project

```sh
git clone https://github.com/vientorepublic/DiscordChatRelay.git
```

## install dependencies

```sh
npm install
```

## build

```sh
npm run build
```

## start bot

```sh
npm run start
```
